# ความปลอดภัย (Security & Safety)

เอกสารนี้อธิบายว่าโปรแกรมทำอะไร/ไม่ทำอะไร เพื่อให้คนใช้มั่นใจว่า **ปลอดภัยต่อ
เครื่อง** และ **ปลอดภัยต่อบัญชีเกม** มากที่สุด

## 1. ปลอดภัยต่อเครื่อง (โปรแกรมนี้ไม่ฝังอะไรแปลก ๆ)

- **โอเพนซอร์สทั้งหมด** — โค้ดทุกบรรทัดอ่านได้ ไม่มีไฟล์ที่ถูก obfuscate/เข้ารหัส
- **ไม่มี telemetry / ไม่เก็บข้อมูลผู้ใช้** — ไม่ส่งข้อมูลใด ๆ ออกนอกเครื่อง
- **ต่อเน็ตที่เดียว: `poe.ninja` ผ่าน HTTPS** เพื่อดึงราคาเท่านั้น
  (ตรวจได้ที่ [`client.py`](poe_price/client.py) — มี URL เดียวคือ `poe.ninja`)
- **ไม่มีการดาวน์โหลดแล้วรันโค้ด** — ไม่มี auto-update ที่โหลดไฟล์มารัน,
  ไม่มี `eval`/`exec` ของข้อมูลจากเน็ต, ไม่เรียก process ภายนอก
- **dependency น้อยและตรวจสอบได้:**
  - price core + จับภาพหน้าจอ = **ไม่มี dependency เลย** (Python standard library + ctypes)
  - OCR = ใช้ `winrt-*` ซึ่งเป็น **binding ทางการของ Microsoft** เรียก OCR ที่มากับ Windows
  - GUI = `tkinter` (มากับ Python)
- **ถ้าจะให้ชัวร์ที่สุด:** รันจากซอร์สเอง (`py -m poe_price.app`) แทนการรับ .exe จากคนอื่น
  หรือ build .exe เองจากซอร์สที่อ่านแล้ว

## 2. ปลอดภัยต่อบัญชีเกม (ลดความเสี่ยงโดนแบน)

โปรแกรมนี้อยู่ในกลุ่ม **"overlay อ่านหน้าจอ"** ซึ่งเป็นกลุ่มที่เสี่ยงต่ำที่สุด
เพราะ **ไม่แตะตัวเกมเลย**:

| สิ่งที่โปรแกรม **ทำ** | สิ่งที่โปรแกรม **ไม่ทำ** (สำคัญ) |
|---|---|
| จับ "ภาพหน้าจอ" (เหมือนกด print screen) | ❌ ไม่อ่าน/เขียนหน่วยความจำของเกม |
| อ่านตัวอักษรจากภาพด้วย OCR | ❌ ไม่ inject DLL / ไม่ hook เข้า process เกม |
| ขอราคาจาก poe.ninja | ❌ ไม่ส่งคีย์/คลิกเข้าเกมแทนผู้เล่น (no automation) |
| วาดข้อความทับบนจอ (คลิกทะลุ) | ❌ ไม่ดักหรือแก้แพ็กเก็ตเครือข่ายของเกม |

หลักการ: โปรแกรม **อ่านอย่างเดียว ฝั่งนอกเกม** ไม่มีการ "ทำแทน" ผู้เล่นและไม่ยุ่งกับ
ไฟล์/หน่วยความจำ/เครือข่ายของเกม — เป็นแนวทางเดียวกับเครื่องมือเช็คราคาทั่วไป

> ⚠️ **ข้อพึงระวังที่ซื่อสัตย์:** ผู้พัฒนาเกม (GGG) เป็นผู้ตัดสินใจสุดท้ายว่าอนุญาต
> เครื่องมือใด การใช้ third-party tool ใด ๆ มีความเสี่ยงตามกฎของเกมเสมอ แม้กลุ่ม
> "overlay อ่านหน้าจอ" จะถือว่าเสี่ยงต่ำ ผู้ใช้ควรอ่าน ToS ปัจจุบันและรับความเสี่ยงเอง

## 3. ขอบเขตสิทธิ์ที่โปรแกรมใช้

- จับภาพหน้าจอ (GDI `BitBlt`) — สิทธิ์ผู้ใช้ปกติ ไม่ต้องเป็น admin
- global hotkey (`RegisterHotKey`) — เห็น **เฉพาะปุ่มที่ลงทะเบียน** (F5/F4/Ctrl+Alt+Q)
  ไม่ใช่ keylogger; ไม่ได้ใช้ keyboard hook ระดับล่างที่อ่านทุกปุ่ม
- หน้าต่าง overlay ตั้ง `WS_EX_TRANSPARENT` = **คลิกทะลุทุกจุด** จึงไม่มีทางไปบัง/ดัก
  คลิกของเกมได้ และ `WS_EX_NOACTIVATE` = ไม่แย่งโฟกัสจากเกม
